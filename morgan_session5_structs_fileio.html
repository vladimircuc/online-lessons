Structures from Scratch

A struct lets you bundle multiple variables into one custom type. Think of it as creating your own data type.

The Problem Structs Solve

Say you want to store info about a student: their name, age, and GPA. Without structs, you'd need three separate variables:

string name = "Morgan"; int age = 20; double gpa = 3.5;

Now imagine you need 30 students. You'd need 3 arrays (one for names, one for ages, one for GPAs) and keep them in sync. That's a mess.

A struct lets you bundle all three into ONE thing:

struct Student { string name; int age; double gpa; };

Now Student is a new data type you invented. Like how int holds a number, Student holds a name + age + gpa together.

Using a Struct

struct Student { string name; int age; double gpa; }; int main() { Student s1; // create a Student variable s1.name = "Morgan"; // use the DOT to access each piece s1.age = 20; s1.gpa = 3.5; cout << s1.name << " is " << s1.age << endl; return 0; }
💡 The Dot Operator

Use a dot (.) to access members of a struct. s1.name means "go into the s1 variable and get the name part."

Reading Struct Members from User

Student s1; cout << "Enter name: "; getline(cin, s1.name); // for strings with spaces cout << "Enter age: "; cin >> s1.age; cout << "Enter GPA: "; cin >> s1.gpa;

Nested Structs & Arrays

Putting a struct inside a struct, and making arrays of structs.

Nested Structures

A struct member can be another struct. For example, a Flight has a Date:

struct Date { int month, day, year; }; struct Flight { string flightNo; string source; string destination; Date date; // a struct INSIDE a struct int numPassengers; };

To access the nested member, use two dots:

Flight f; f.flightNo = "DL 110"; f.date.month = 12; // f.date gets the Date, .month gets the month inside it f.date.day = 25; f.date.year = 2026;

Arrays of Structures

Just like arrays of ints, you can have arrays of your custom type:

Student roster[30]; // array of 30 Students // access the 3rd student's name: roster[2].name = "Alex"; // loop through all students: for(int i = 0; i < 30; i++) cout << roster[i].name << ": " << roster[i].gpa << endl;

Part 3 / 10

Structs & Functions

Passing structs to functions and returning them.

Passing a Struct to a Function

You can pass structs by value (copy) or by reference (&), just like regular variables:

// Pass by value: works on a COPY, can't change original void print(Student s) { cout << s.name << ", GPA: " << s.gpa << endl; } // Pass by reference: works on the ORIGINAL void read(Student &s) { cout << "Name: "; getline(cin, s.name); cout << "Age: "; cin >> s.age; cout << "GPA: "; cin >> s.gpa; }

Passing an Array of Structs

Same rule as regular arrays: arrays are always passed by reference automatically, and you need to send the size too.

void readAll(Student arr[], int size) { for(int i = 0; i < size; i++) { cout << "Enter name: "; getline(cin, arr[i].name); cout << "Enter GPA: "; cin >> arr[i].gpa; cin.ignore(); // clear newline before next getline } }
⚠️ cin.ignore() Reminder

When you mix cin >> with getline, the cin >> leaves a newline character behind. cin.ignore() eats that leftover newline so the next getline works correctly.

Part 4 / 10

Dynamic Structs & Pointers

Creating structs with new, dynamic arrays of structs, and the arrow operator.

Dynamic Arrays of Structs

Same idea as dynamic int arrays, just with your struct type:

int n; cin >> n; Student* roster = new Student[n]; // dynamic array of n Students for(int i = 0; i < n; i++) { cout << "Name: "; getline(cin, roster[i].name); cout << "GPA: "; cin >> roster[i].gpa; cin.ignore(); } delete[] roster; // free when done

Pointer to a Single Struct

Student* ptr = new Student; (*ptr).name = "Morgan"; // parentheses needed! * before dot (*ptr).gpa = 3.5;

That (*ptr).name syntax is ugly. C++ gives you a shortcut:

⭐ The Arrow Operator (->)
Student* ptr = new Student; ptr->name = "Morgan"; // same as (*ptr).name ptr->gpa = 3.5; // same as (*ptr).gpa delete ptr;

ptr->member is identical to (*ptr).member. The arrow is just cleaner.

Part 5 / 10

File I/O Basics

Reading from and writing to files using ifstream and ofstream.

The Setup

#include<fstream> // this header gives you ifstream and ofstream
Reading (ifstream)
ifstream inFile; inFile.open("data.txt"); if(!inFile) // check if it opened { cout << "Error!" << endl; return 0; } int x; inFile >> x; // read like cin inFile.close(); // close when done
Writing (ofstream)
ofstream outFile; outFile.open("output.txt"); outFile << "Hello!" << endl; outFile << 42 << endl; outFile.close();
💡 Key Insight

ifstream works just like cin but reads from a file instead of the keyboard.

ofstream works just like cout but writes to a file instead of the screen.

Same >> and << operators, same getline, just a different source/destination.

Reading All Data from a File

Pattern 1: You know how many items

// file has 200 numbers int arr[200]; for(int i = 0; i < 200; i++) inFile >> arr[i];

Pattern 2: Unknown count, read until end

// file has unknown number of doubles, one per line double val; double sum = 0; int count = 0; while(inFile >> val) // returns false when file ends { sum += val; count++; } cout << "Average: " << sum / count << endl;

Pattern 3: Reading lines with getline

string line; while(getline(inFile, line)) // read one full line at a time { cout << line << endl; }

Part 6 / 10

Reading Formatted Input

Handling slashes, colons, and other separators when reading from user or file.

The Trick: Use a char to Eat Separators

When input looks like 04/20/2026 13:20, you can't just cin >> month >> day >> year because the slashes are in the way. Use a char variable to consume them:

int month, day, year, hour, minute; char slash, colon; cout << "Enter date and time: "; cin >> month >> slash >> day >> slash >> year >> hour >> colon >> minute; // Input: 04/20/2026 13:20 // month=4, slash='/', day=20, slash='/', year=2026, hour=13, colon=':', minute=20

The char slash variable reads the '/' character and throws it away. Same with colon for ':'. The important numbers end up in the right int variables.

MCQ

To read input formatted as 12-25-2026 into month, day, year, what goes between cin >> month and >> day?

Counting Words in a Line

A common exam pattern: read a line with getline, then count words by counting spaces.

string line; getline(inFile, line); int words = 0; if(line.length() > 0) words = 1; // at least one word if line isn't empty for(int i = 0; i < line.length(); i++) { if(line[i] == ' ') words++; }

Logic: number of words = number of spaces + 1 (assuming no leading/trailing/double spaces).

Problem 4: Perfect Numbers (Study Guide Q4)

A perfect number equals the sum of its factors (not including itself). Example: 6 = 1 + 2 + 3. Write a program to print all perfect numbers between 2 and 1000.

Expected output: 6, 28, 496

#include<iostream> using namespace std; int main() { for(int num = 2; num <= 1000; num++) { int sumFactors = 0; for(int i = 1; i < num; i++) { if(num % i == 0) sumFactors += i; } if(num == sumFactors) cout << num << endl; } return 0; }
Key Points

Outer loop goes through each candidate number. Inner loop checks all possible factors (1 to num-1) using the modulo operator. If num % i == 0, then i is a factor. Compare the sum to the number itself.

Problem 5: File Average (from exam materials)

Write a program that reads from a file called "lines.txt". This file contains an unknown number of doubles, one per line. Find the average and print it.

#include<iostream> #include<fstream> using namespace std; int main() { ifstream inFile; inFile.open("lines.txt"); if(!inFile) { cout << "Error opening file" << endl; return 0; } double val, sum = 0; int count = 0; while(inFile >> val) { sum += val; count++; } inFile.close(); if(count > 0) cout << "Average: " << sum / count << endl; else cout << "File was empty" << endl; return 0; }

Problem 6: Print Duplicates (Study Guide Q5)

Write a program that creates an array of 200 integers and reads the values from "input.txt". Then print all numbers that occur more than once. You can assume the file opens fine and has exactly 200 numbers. It's OK if a duplicate prints multiple times.

#include<iostream> #include<fstream> using namespace std; int main() { ifstream inFile; int arr[200]; inFile.open("input.txt"); if(!inFile) { cout << "Error opening file" << endl; return 0; } for(int i = 0; i < 200; i++) inFile >> arr[i]; inFile.close(); // check each element against every other for(int i = 0; i < 200; i++) { for(int j = 0; j < 200; j++) { if(i != j && arr[i] == arr[j]) { cout << arr[i] << endl; break; // move to next i after first match } } } return 0; }
Key Points

Nested loop: for each element, check every other element. i != j avoids comparing an element to itself. The break after the first match avoids printing the same number multiple times for the same i (but the number may still print for different i values, which is OK per the problem statement).

Exam Problem 2: Timestamp Comparison

Write a C++ program to:

  1. Create a structure called Timestamp with 5 integer members: year, month, day, hour, minute.
  2. In main, create a Timestamp and read it from the user in the format: MM/DD/YYYY HH:MM
  3. Check if the object is after 04/19/2026 23:59. Print "Yes" or "No".

Sample: Enter the time: 04/20/2026 13:20 → Yes

#include<iostream> using namespace std; struct Timestamp { int year, month, day, hour, minute; }; int main() { Timestamp ts; char slash, colon; cout << "Enter the time: "; cin >> ts.month >> slash >> ts.day >> slash >> ts.year >> ts.hour >> colon >> ts.minute; bool after = false; if(ts.year > 2026) after = true; else if(ts.year == 2026) { if(ts.month > 4) after = true; else if(ts.month == 4) { if(ts.day > 19) after = true; else if(ts.day == 19) { if(ts.hour > 23) after = true; else if(ts.hour == 23 && ts.minute > 59) after = true; } } } cout << (after ? "Yes" : "No") << endl; return 0; }
Key Points

Use char variables to eat the / and : separators. Compare from most significant (year) to least significant (minute) using nested if-else. Once a higher field is definitively greater, set after = true without checking lower fields.

Exam Problem 3: Dynamic Struct Array with Search

Write a C++ program that creates a structure called Node with 2 members: an integer called key and a string called value. Read in the number of elements from the user, then create a dynamic array of Nodes for that size. Read in all values. Then write a function called search that accepts the array, its size, and an integer to search for. It implements linear search and returns the value part of the matching Node. If not found, return "not found". Call the function and print the result.

#include<iostream> #include<string> using namespace std; struct Node { int key; string value; }; string search(Node* arr, int size, int target); int main() { int n; cout << "How many elements? "; cin >> n; cin.ignore(); Node* arr = new Node[n]; for(int i = 0; i < n; i++) { cout << "Enter key: "; cin >> arr[i].key; cin.ignore(); cout << "Enter value: "; getline(cin, arr[i].value); } int searchKey; cout << "Search for key: "; cin >> searchKey; cout << search(arr, n, searchKey) << endl; delete[] arr; return 0; } string search(Node* arr, int size, int target) { for(int i = 0; i < size; i++) { if(arr[i].key == target) return arr[i].value; } return "not found"; }
Key Points

This combines dynamic memory (new Node[n]), structs, cin.ignore() between cin >> and getline, a search function with arrays of structs, and delete[] cleanup. This is as hard as exam problems get for this course.

Exam Problem 4: File Word Count Per Line

Write a C++ program that reads from a file called "SomeText.txt". The first line tells you how many lines follow. Use getline to read each line and find the number of words in each line. Print the word count for each line.

#include<iostream> #include<fstream> #include<string> using namespace std; int main() { ifstream inFile; inFile.open("SomeText.txt"); if(!inFile) { cout << "Error opening file" << endl; return 0; } int numLines; inFile >> numLines; inFile.ignore(); // eat the newline after the number for(int i = 0; i < numLines; i++) { string line; getline(inFile, line); int words = 0; if(line.length() > 0) words = 1; for(int j = 0; j < line.length(); j++) { if(line[j] == ' ') words++; } cout << "Line " << (i + 1) << ": " << words << " words" << endl; } inFile.close(); return 0; }
Key Points

inFile.ignore() after reading the number is critical: without it, the getline on the next iteration would read the leftover newline as an empty line. Word count = spaces + 1 (for non-empty lines).

What's a Reference Variable? (Study Guide Q7)

Short Answer

A reference variable is a formal parameter that acts as a nickname for the actual parameter. Any changes made to the formal parameter will be made to the actual parameter as well.

Example: void change(int &ref); // ref is a reference to whatever int is passed in
