What is an Array?

From complete scratch — what an array is, what an index is, how memory works, and how to declare and access elements.

The Problem Arrays Solve

Imagine storing grades for 5 students. Without arrays you'd need 5 separate variables:

the hard way
int grade1 = 90;
int grade2 = 85;
int grade3 = 72;
int grade4 = 95;
int grade5 = 88;
// What if you had 100 students?? You'd need 100 variables.

With an array, you store them all under one name, accessed by position:

the array way
int grades[5] = {90, 85, 72, 95, 88};
// One name. Five values. Access any of them by index.

The Index — The Most Important Concept

💡 The Golden Rule
Arrays in C++ are zero-indexed. The first element is always at index 0, not 1. The last element is always at index size - 1.

Think of an array like a row of labeled boxes. Each box holds a value. The label on the box is the index:

value

Array name: grades. Size: 5. Index runs from 0 to 4 (size - 1 = 4). Highlighted: last element.

accessing elements
int grades[5] = {90, 85, 72, 95, 88};

cout << grades[0];    // 90  — first element
cout << grades[2];    // 72  — third element (index 2)
cout << grades[4];    // 88  — last element (index = size-1 = 4)

// Changing a value
grades[1] = 100;      // now grades[1] = 100 instead of 85

// NEVER do this — goes out of bounds!
cout << grades[5];    // DANGER: array only has indices 0-4

How to Declare an Array

declaring_arrays.cpp
// Syntax: type  name[size];
int     arr[10];           // 10 integers — contains garbage until assigned
double  scores[15];        // 15 doubles
float   temps[5];          // 5 floats
char    letters[26];       // 26 characters

// Declare AND initialize at the same time
int arr[5] = {3, -5, 14, 100, -67};

// Partial init — remaining slots become 0
int arr[10] = {1, 2, 3};    // arr[3] through arr[9] are all 0

// Size must be a constant — you can't use a regular variable as size
const int SIZE = 10;
int arr[SIZE];               // this is fine
int n = 10;
int arr[n];                  // this may NOT work in all compilers — avoid it

Array Memory — How the Address Formula Works

In memory, array elements sit right next to each other. That's why we can calculate any element's exact address:

💡 Address Formula
Address = Base_Address + sizeof(type) × index

sizeof values: int=4, double=8, float=4, short=2, char=1
address_calculation.cpp
// Example: int array starting at address 1000
// int arr[5] = {90, 85, 72, 95, 88};
// sizeof(int) = 4

// arr[0] lives at: 1000 + 4 × 0 = 1000
// arr[1] lives at: 1000 + 4 × 1 = 1004
// arr[2] lives at: 1000 + 4 × 2 = 1008
// arr[3] lives at: 1000 + 4 × 3 = 1012
// arr[4] lives at: 1000 + 4 × 4 = 1016

// Each int takes 4 bytes, so elements are 4 apart in memory

Reading and Printing an Array

⭐ Professor's Exact Pattern — Memorize This
Read with a for loop using cin >> arr[i]. Print with a for loop using arr[i] << "\t". One endl after the loop. This exact pattern appears on every exam.
read_and_print.cpp
#include<iostream>
using namespace std;

int main()
{
    int arr[5];

    // Reading from user — loop through each index
    cout << "Enter 5 values: ";
    for(int i=0; i<5; i++)
        cin >> arr[i];      // reads one value into arr[i] each time

    // Printing — tab between each element, endl after loop
    cout << "The array is: ";
    for(int i=0; i<5; i++)
        cout << arr[i] << "\t";   // \t = tab character
    cout << endl;                  // new line after the whole array

    return 0;
}

Looping Through an Array

common_loops.cpp
int arr[5] = {4, -2, 7, 0, 3};

// Sum all elements
int sum = 0;
for(int i=0; i<5; i++)
    sum += arr[i];           // sum = sum + arr[i]
cout << "Sum: " << sum << endl;    // Sum: 12

// Count negatives
int negCount = 0;
for(int i=0; i<5; i++)
{
    if(arr[i] < 0)
        negCount++;
}
cout << "Negatives: " << negCount << endl;    // Negatives: 1

// Find the maximum
int max = arr[0];            // assume first is max
for(int i=1; i<5; i++)      // start at 1 since we already grabbed arr[0]
{
    if(arr[i] > max)
        max = arr[i];        // update if found something bigger
}
cout << "Max: " << max << endl;    // Max: 7

// Linear search — find index of a value
int key = 7;
int found = -1;              // -1 means not found yet
for(int i=0; i<5; i++)
{
    if(arr[i] == key)
    {
        found = i;           // store the index where we found it
        break;               // stop searching once found
    }
}
if(found == -1)
    cout << "Not found" << endl;
else
    cout << "Found at index " << found << endl;    // Found at index 2

Modifying Array Elements

modifying.cpp
int arr[5] = {10, 20, 30, 40, 50};

// Double every element
for(int i=0; i<5; i++)
    arr[i] = arr[i] * 2;
// arr is now: {20, 40, 60, 80, 100}

// Replace every even element with 0
for(int i=0; i<5; i++)
{
    if(arr[i] % 2 == 0)
        arr[i] = 0;
}

// CAREFUL: when replacing elements using neighbors, copy first!
int arr2[5] = {1, 2, 3, 4, 5};
int temp[5];
for(int i=0; i<5; i++)
    temp[i] = arr2[i];       // copy original values

for(int i=0; i<4; i++)
    arr2[i] = temp[i] + temp[i+1];   // use temp, not arr2 (which is changing!)
// If you used arr2[i] + arr2[i+1], arr2[i] would already be modified!
⚠ The Temp Array Trap
Whenever you need to replace an element using its neighbors, always copy to a temp array first. Otherwise element 0 changes, then when you calculate element 1 using element 0 you get the wrong (new) value. This is why sumReplace, moduli, and alternates all need a temp copy.

Arrays + Functions

How to pass arrays to functions, the professor's required format, and how to write the array functions that appear on the exam.

⭐ The Exam Pattern — Read This First
The code writing question on the exam is always this same pattern: read array in main → pass to function → function modifies or computes → print result in main. If you can do this in your sleep, you pass.

How Passing an Array Works

When you pass an array to a function, you're NOT copying the whole array. The function gets a direct connection to the original. Any change inside the function changes the real array. No & needed — it happens automatically.

passing_arrays.cpp
// The parameter: type name[], int size
// Always pass size as a second parameter — the function has no other way to know it

void doubleAll(int arr[], int size)
{
    for(int i=0; i<size; i++)
        arr[i] = arr[i] * 2;    // changes the REAL array in main
}

// Use const when the function should NOT modify the array
int findMax(const int arr[], int size)
{
    int max = arr[0];
    for(int i=1; i<size; i++)
        if(arr[i] > max) max = arr[i];
    return max;
}

The Full Professor-Style Program

full_pattern.cpp — the exam template
#include<iostream>
using namespace std;

// Step 1: Forward declarations at top
void myFunction(int arr[], int size);
int computeSomething(const int arr[], int size);

}

int computeSomething(const int arr[], int size)
{
    int sum = 0;
    for(int i=0; i<size; i++)
        sum += arr[i];
    return sum;
}

sumReplace.cpp
void sumReplace(int arr[], int size)
{
    // MUST copy first — otherwise changing arr[0] corrupts arr[1]'s calculation
    int temp[10];
    for(int i=0; i<size; i++)
        temp[i] = arr[i];

    for(int i=0; i<size; i++)
    {
        // Ternary operator: condition ? valueIfTrue : valueIfFalse
        int left  = (i == 0)       ? -3  : temp[i-1];    // edge case: position 0
        int right = (i == size-1)  ? 250 : temp[i+1];    // edge case: last position

        arr[i] = left + right;    // replace with sum of neighbors
    }
}

alternates — every 3rd element

alternates.cpp
void alternates(float arr[], int size)
{
    // Every 3rd element = index 2, 5, 8, 11...
    // Start at i=2, step by 3 each time
    for(int i=2; i<size; i+=3)
    {
        arr[i] = arr[i] + arr[i-1] + arr[i-2];
        // arr[2] = arr[2] + arr[1] + arr[0]
        // arr[5] = arr[5] + arr[4] + arr[3]  etc.
    }
    // No temp needed here because we only replace i=2, 5, 8...
    // We read i-1 and i-2 which we haven't modified yet
}

🔍 Practice 2 — Write moduli from scratch

Write void moduli(int arr[], int size): replace each element with arr[i] % arr[i+1]. If arr[i+1] == 0, leave the element unchanged.

moduli.cpp
void moduli(int arr[], int size)
{
    int temp[15];
    for(int i=0; i<size; i++)
        temp[i] = arr[i];       // copy so we use original values

    for(int i=0; i<size-1; i++)    // stop at size-2 (need i+1 to exist)
    {
        if(temp[i+1] == 0)
            arr[i] = temp[i];       // leave unchanged if divisor is 0
        else
            arr[i] = temp[i] % temp[i+1];
    }
    // arr[size-1] (last element): no i+1 exists, leave unchanged
}

Nested Loops — The Only Way to Traverse 2D Arrays

nested_loops.cpp
int mat[3][3] = {{1,5,19},{6,-2,10},{12,8,5}};

// Print the whole 2D array
for(int i=0; i<3; i++)         // i = row number
{
    for(int j=0; j<3; j++)     // j = column number
    {
        cout << mat[i][j] << " ";   // print each cell
    }
    cout << endl;               // new line after each complete row
}
// Output:
// 1 5 19
// 6 -2 10
// 12 8 5

// Sum of the MAIN DIAGONAL (where row == col: mat[0][0], mat[1][1], mat[2][2])
int sum = 0;
for(int i=0; i<3; i++)
    sum += mat[i][i];           // i used for BOTH row and col
// 1 + (-2) + 5 = 4

// Sum of ALL elements
int total = 0;
for(int i=0; i<3; i++)
    for(int j=0; j<3; j++)
        total += mat[i][j];

Passing 2D Arrays to Functions

⚠ Important — Column Count is Required
When passing a 2D array to a function, you MUST write the column count in the parameter. The row count can be a variable, but the column count must be hard-coded. This is just how C++ works with 2D arrays.
2d_functions.cpp
// Column count MUST be specified in the parameter (9 here)
void printArray(int arr[][9], int rows)
{
    for(int i=0; i<rows; i++)
    {
        for(int j=0; j<rows; j++)    // or use a cols parameter
            cout << arr[i][j] << " ";
        cout << endl;
    }
}

// Filling a 2D array
void fillZeros(int arr[][9], int rows)
{
    for(int i=0; i<rows; i++)
        for(int j=0; j<rows; j++)
            arr[i][j] = 0;
}

int main()
{
    int mat[9][9];
    fillZeros(mat, 9);      // pass name (no brackets needed) + rows
    printArray(mat, 9);
    return 0;
}

C-Strings

A C-string is just an array of chars with a special ending character. Same array concepts, new type.

A C-String is an Array of Characters

Everything you learned about arrays applies here. The only difference: C-strings end with a special character called the null terminator ('\0'). It marks where the string ends.

💡 The Null Terminator
Every C-string has a hidden '\0' at the end. That's how functions like strlen know where the string ends — they loop until they hit '\0'. Always declare your char array a bit bigger than you need to make room for it.

str2 = str1 is a compile error. str1 == str2 compares memory addresses, not content. Always use the library functions.

cstring_library.cpp
#include<iostream>
#include<cstring>
using namespace std;

int main()
{
    char s1[100] = "Hello";
    char s2[100] = "World";
    char s3[100];

    // Length
    cout << strlen(s1);          // 5

    // Copy — s3 now contains "Hello"
    strcpy(s3, s1);

    // Concatenate — s1 is now "HelloWorld"
    strcat(s1, s2);
    cout << s1;                   // HelloWorld

    // Compare
    if(strcmp(s3, s2) == 0)
        cout << "Same" << endl;
    else
        cout << "Different" << endl;    // Different

    return 0;
}

🔍 Practice 2 — Write from scratch

Write a complete program with a function countVowels that takes a C-string and returns how many vowels (a,e,i,o,u — upper or lower case) it contains. Read a string in main, call the function, print the count.

countVowels.cpp
#include<iostream>
#include<cstring>
#include<cctype>
using namespace std;

int countVowels(char str[]);

int main()
{
    char s[200];
    cout << "Enter a string: ";
    cin.getline(s, 200);
    cout << "Vowels: " << countVowels(s) << endl;
    return 0;
}

int countVowels(char str[])
{
    int count = 0;
    for(int i=0; i<strlen(str); i++)
    {
        char c = tolower(str[i]);    // convert to lowercase to check either case
        if(c=='a' || c=='e' || c=='i' || c=='o' || c=='u')
            count++;
    }
    return count;
}

string_objects.cpp
#include<iostream>
#include<string>
using namespace std;

int main()
{
    string s1 = "Hello";
    string s2 = "World";
    string s3;

    // Length (both do the same thing)
    cout << s1.length();        // 5
    cout << s1.size();          // 5

    // Concatenation — two ways
    s3 = s1 + " " + s2;         // s3 = "Hello World"
    s1 += "!";                   // s1 = "Hello!"

    // Comparison — works directly, just like numbers
    if(s1 == s2) cout << "Same" << endl;
    if(s1 != s2) cout << "Different" << endl;
    if(s1 < s2)  cout << "s1 comes first alphabetically" << endl;

    // Access individual characters — same as arrays
    cout << s3[0];               // 'H'
    cout << s3.at(6);            // 'W' — .at() with bounds check

    // Find a substring — returns index, or string::npos if not found
    int pos = s3.find("World");  // pos = 6
    if(pos != string::npos)
        cout << "Found at " << pos << endl;

    // Substring — .substr(startIndex, length)
    string sub = s3.substr(6, 5);   // "World" (start=6, length=5)

    // Reading
    string word;
    cin >> word;                     // reads one word
    string line;
    getline(cin, line);              // reads whole line (note: NOT cin.getline)

    return 0;
}

changeE_solution.cpp — professor's style
#include<iostream>
#include<cstring>
using namespace std;

void changeE(char str[]);    // forward declaration

int main()
{
    char st[200];
    cout << "Input:\n";
    cin.getline(st, 200);    // read full line — cin >> would stop at space
    changeE(st);             // call function — modifies st directly
    cout << "Output:\n" << st << endl;
    return 0;
}

void changeE(char str[])
{
    for(int i=0; i<strlen(str); i++)   // loop from 0 to end of string
    {
        if(str[i] == 'e')              // check for lowercase 'e' only
            str[i] = '3';             // replace in place
    }
}

Problem 5 — sumSquares (array + function) Similar to 25pt exam problem

Write a function sumSquares that accepts a double array and its size. The array represents side lengths of squares. Replace each value with its square (area), and return the sum of the squares. In main: create array of 5 doubles, read values, call the function, print the array and the sum.

Sample Run

The array is:
156.25   94.09   16   6.9169   50.41
The sum is 323.667
sumSquares_solution.cpp
#include<iostream>
using namespace std;

double sumSquares(double arr[], int size);    // forward declaration

int main()
{
    double arr[5];

    // Read array
    cout << "Enter the array values:\n";
    for(int i=0; i<5; i++)
        cin >> arr[i];

    // Call function — it modifies arr AND returns the sum
    double ans = sumSquares(arr, 5);

    // Print modified array (professor's tab style)
    cout << "The array is:\n";
    for(int i=0; i<5; i++)
        cout << arr[i] << "\t";
    cout << endl;

    // Print the returned sum
    cout << "The sum is " << ans << endl;

    return 0;
}

double sumSquares(double arr[], int size)
{
    double sum = 0;
    for(int i=0; i<size; i++)
    {
        arr[i] = arr[i] * arr[i];   // replace value with its square
        sum = sum + arr[i];          // add the square to running sum
    }
    return sum;                      // return total sum
}
💡 Notice — No temp copy needed here
We don't need a temp array because we replace arr[i] using only arr[i] itself. We're not looking at neighbors. If we needed arr[i-1] or arr[i+1], we'd need a copy first.

Write a function that receives a 2D int array, number of rows, and a starting number. Populate it in a triangle pattern starting from the center of the top row, growing by 1 wider each row. Empty spots filled with 0.

⚠ This is the Most Complex Problem
Don't panic if this seems hard. On the exam, turn in whatever you have — there is a lot of partial credit. Write the main function and at least the loop structure even if the logic isn't perfect.
fillArray_solution.cpp
#include<iostream>
using namespace std;

void fillArray(int arr[][9], int rows, int num);

int main()
{
    int arr[9][9], start, rows;
    cout << "Enter the number of rows: ";
    cin >> rows;
    cout << "Enter the first value: ";
    cin >> start;

    fillArray(arr, rows, start);

    cout << "The array is \n";
    for(int i=0; i<rows; i++)
    {
        for(int j=0; j<rows; j++)
            cout << arr[i][j] << " ";
        cout << endl;
    }
    return 0;
}

void fillArray(int arr[][9], int rows, int num)
{
    // zeros = how many leading zeros in this row
    // starts at rows/2 (for 9 rows: 4 zeros at start)
    int zeros = rows / 2;

    for(int i=0; i<rows; i++)
    {
        int j = 0;

        // Fill leading zeros
        for(j=0; j<zeros; j++)
            arr[i][j] = 0;

        // Fill the actual numbers (triangle grows: row i has i+1 numbers)
        for(int k=0; j<=i+zeros; k++)
        {
            arr[i][j] = num;
            num++;
            j++;
        }

        // Fill trailing zeros
        for(; j<rows; j++)
            arr[i][j] = 0;

        // Every other row (even rows) the triangle gets 1 wider on left
        if(i % 2 == 0)
            zeros--;
    }
}
